Bem-vindo ao Gerador de Grade Horária

Este sistema foi projetado para ajudá-lo a criar grades horárias para escolas de forma eficiente, utilizando um algoritmo genético para otimizar o processo e atender às restrições dos professores e turmas.

Como Usar

  1. Vá para a página Cadastro de Turmas, para adicionar todas as turmas da sua escola.
  2. Em seguida, acesse Professores e Cargas, para cadastrar os professores e suas respectivas cargas horárias.
  3. Por fim, clique em Grade Horária para gerar a grade automaticamente.
